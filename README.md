Automation Algorithm:
1. Find the theta value of the angle between frame line and detected center line
2. Divide the value by 180, return a percentage
3. Get the other percentage(1 - inital percentage)
4. The original detected percentage x 70 = left motor value
5. The other percentage x 70 = right motor value
